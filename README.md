# ElevenLabs Voice AI Showcase

Portfolio site presenting four voice AI demos built on ElevenLabs Conversational AI, spanning telecom, food ordering, education, and healthcare.

**Live:** [elevenlabsagent-showcase.vercel.app](https://elevenlabsagent-showcase.vercel.app)

## Demos

| Demo | Vertical | Live URL |
|------|----------|----------|
| Airtel AI Support | Telecom | [elevenlabsagent-airtel.vercel.app](https://elevenlabsagent-airtel.vercel.app) |
| Domino's Voice Agent | Quick Service | [elevenlabsagent-dominos.vercel.app](https://elevenlabsagent-dominos.vercel.app) |
| Ringaa | Education | [elevenlabsagent-ringaa.vercel.app](https://elevenlabsagent-ringaa.vercel.app) |
| Meddo | Healthcare | [elevenlabsagent-meddo.vercel.app](https://elevenlabsagent-meddo.vercel.app) |

## Tech Stack

Next.js 16 | TypeScript | Tailwind v4 | Vercel

## Run Locally

```bash
npm install
npm run dev
```

Opens at [http://localhost:3000](http://localhost:3000).
